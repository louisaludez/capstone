### Concierge Module

#### GET `/concierge/` (dashboard)
- Renders main concierge dashboard.

#### GET `/concierge/book-tours/`
- Renders booking UI for tours.

#### GET `/concierge/book-reservations/`
- Renders booking UI for reservations.

#### GET `/concierge/messenger/`
- Renders messenger filtered by role pair.

#### GET `/concierge/timeline/`
- Renders timeline page.
